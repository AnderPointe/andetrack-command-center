# Phase 20 — V3.5 Commercial Ecosystem Expansion (plan)

Phase 20 extends V3 mobile-native expansion into commercial ecosystem
monetization and certification execution.

## In scope (planned)
- Carrier marketplace monetization (transaction fees, settlement workflow)
- Advanced telematics (HOS sync, fuel + idle analytics, driver scorecards)
- Compliance automation (DOT/FMCSA placeholders → real ingest)
- Enterprise certification execution (SOC 2 Type I evidence collection)
- Strategic partnership readiness (3PL, broker, OEM telematics)
- Native driver app production hardening + crash reporting
- Android Auto + CarPlay implementation milestones (still gated on platform approval)
- ML feedback loop for CoPilot recommendations
- White-label native driver builds (per-customer)
- Multi-region GPS sharding pilot

## Deferred to V4
- Fully autonomous dispatch
- Cross-carrier optimization
- Predictive maintenance from telematics ML
- Global tax / customs automation
- Insurance underwriting
