# Phase 19 — V3 plan

V3 extends Anderoute V2.5 enterprise into mobile-native, voice-first, and
marketplace territory.

## In scope
- Mobile-native EliteNav (Expo / React Native shell beyond PWA)
- Android Auto integration (turn-by-turn projection, voice prompts)
- CarPlay integration (turn-by-turn projection, voice prompts)
- Advanced AI voice (conversational CoPilot in-cab, multi-intent)
- Telematics integration (ELD / J1939 / OEM APIs for fuel, fault codes, HOS)
- Carrier marketplace (load posting, bidding, settlement)
- Enterprise certification readiness (SOC 2 Type II evidence automation, ISO 27001 mapping)
- Multi-region read replicas + GPS stream sharding (1,000+ driver scale)
- White-label mobile builds (per-customer branded driver app)
- ML feedback loop (recommendation acceptance / rejection signal)

## Deferred to V4
- Fully autonomous dispatch
- Cross-carrier optimization
- Predictive maintenance from telematics ML
- Global tax / customs automation
